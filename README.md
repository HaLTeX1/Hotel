
# StaySync - Szállásfoglaló rendszer

A program egy terminal alapú szállásfoglaló rendszert valósít meg. A program a Gábor Dénes Egyetem Objektumorientált Programozás tárgyára készült.

# A Program funkciói

A program lehetővé teszi egy fiktív hotelbe való szállásfoglalást, illetve a foglalások kezelését. 
- Foglalás 
- Foglalás törlése
- Foglalások listázása
- Adatlekérdezés (Foglalás ID alapján)
